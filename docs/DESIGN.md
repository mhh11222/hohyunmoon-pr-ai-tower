# MOON AI TOWER — 설계문서 (office-hours, Builder mode)

작성: 2026-06-26 · 작성자: 문호현 + Claude · 승인 대기.
별개 프로젝트(VELVOID 멘토 레포와 분리). 위치 `~/Desktop/moon-ai-tower/`. 배포 = GitHub Pages(정적).

## 비전 / "와" 순간
일반 포트폴리오가 아니다. **방문자가 들어오면 살아 움직이는 오로라 입자장**을 만난다 — 그게 곧 문호현이 실제로 만든 **유전 알고리즘 진화엔진**이다. 스크롤/탐험하면 세대가 진화하고, 파레토 전선이 빛나고, 챔피언 타워(MOON AI TOWER)가 솟고, 각 GitHub 프로젝트는 날아가서 보는 별이며, 본인 얼굴조차 같은 입자로 구성된다. **세상 어떤 포트폴리오에도 없는 한 장면 = 본인의 진짜 엔진이 살아있는 것.**

## 채택안: C "사이트가 곧 GA" (단계적 출시)
하나의 입자 시뮬레이션이 전체 페이지. 섹션 = 그 필드의 영역들. 단, 미출시 괴물 방지를 위해 시네마틱 코어부터 출시 후 점진 진화.

## 기술 스택 (borealis 역설계 반영)
- **Three.js r0.184** (CDN importmap) + 순수 HTML/CSS/JS. **React/Vite 없음** — 로컬에서 `index.html` 열어 바로 확인.
- 히어로 = **`THREE.Points` + 커스텀 `ShaderMaterial`**(가산혼합 글로우 스프라이트, 싸인 노이즈 오로라 컬러, 정점셰이더에 마우스 크레이터·인트로 러시인). 블룸 패스 불필요.
- 스크롤 모션 = 네이티브 스크롤 → 셰이더 uniform 구동(회전·heat·tint). (framer-motion 대신 가벼운 IntersectionObserver/스크롤 진행률.)
- 폰트: **JetBrains Mono**(본문·HUD) + 디스플레이용 1종(무료 대체: Space Grotesk/Unbounded 등, 라이선스 안전한 것). 셀프호스팅.
- 팔레트: 순흑 `#000` · 오로라 시안 `#5eeaff` · 써멀 오렌지 `#ff6a3d` · 오프화이트 `#f4f3ee`.

## 데이터 → 비주얼 매핑 (GA 시각화 = 주인공)
| GA 개념 | 비주얼 |
|---|---|
| 유전자 1개 | 입자 1개 |
| 세대(population) | 입자 구름 |
| 목적공간(2축) | 입자 XY 좌표, 적합도 = Z·밝기·크기 |
| 파레토 전선 | 빛나는 시안 곡선(추가 additive Line), 맥동 |
| 챔피언 | 중앙 거대 비콘 + 수직 타워 빔 = **MOON AI TOWER** |
| 챔피언 교체 | 구챔피언 폭발(uExplode) + 신챔피언 러시인(uIntroT) |
| 톰슨 밴딧 탐색 | 해당 위치에 크레이터 펄스(uMouse) |
| 세대 진행 | 입자들이 신좌표로 lerp, 전체가 "숨쉬듯" 재채색 |
데이터원 = Supabase `evo_*`를 **`/data/evo.json`로 export**해 재생(라이브 백엔드 불필요). 초기엔 합성 데이터로 시작 가능.

## 섹션 = 필드의 영역
1. **히어로** — 타워 비콘(챔피언) + 인트로 러시인으로 필드 형성. 이름·한 줄.
2. **GA 진화 (센터피스)** — 세대 애니메이션 + 파레토 전선 맥동. "내 엔진이 살아있다."
3. **프로젝트** — 각 GitHub 레포 = 필드 속 별/유전자. 날아가서(fly-to) 클릭 → 상세 카드(GitHub API: 이름·설명·스타·언어·링크).
4. **About / 입자 초상** — 본인 얼굴이 같은 오로라 입자로 구성, 흩어졌다 재조립.
5. **연락** — 입자 폭발 버스트 → 연락처(이메일·GitHub·기타).

# 디자인 시스템 (design-consultation 확정)

**기억시킬 한 가지:** "그가 만든 AI가 눈앞에서 진화한다" = 살아있는 기계. 모든 결정이 이걸 섬긴다.
**차별화(반-borealis-클론):** globe→수직 타워 / 3D구→2D 목적공간 산점도+파레토 호 / +모노 텔레메트리 HUD / 연속스월→세대 펄스 / 오렌지=발열(적합도) 의미축.

## 컬러 토큰
```css
--ink:#000;            /* 배경 */
--surface:#060A0F;     /* 카드·HUD 패널(순흑 위 살짝) */
--bone:#f4f3ee;        /* 본문 텍스트 */
--mute:#8aa0a8;        /* 보조 텍스트 */
--aurora:#5eeaff;      /* 시그니처 시안 = 링크·강조·활성 */
--aurora-dim:#2a7d8c;  /* 비활성/도미네이티드 */
--pareto:#38f27f;      /* 파레토 전선 그린 */
--thermal:#ff6a3d;     /* 발열=적합도 상승, 챔피언 교체 */
--scrim:rgba(0,0,0,.55); /* 텍스트 가독 플레이트 */
```
대비 보증: 본문은 `--bone`(시안 본문 금지=눈피로/대비), 시안은 강조·링크·라벨만.

## 타이포 (셀프호스팅 OFL)
- 디스플레이: **Space Grotesk** 700/500. 본문·HUD·수치: **JetBrains Mono** 400/700.
- 스케일(1.25 major-third, clamp 유체):
  `--t-hero:clamp(2.8rem,8vw,6.5rem)` / `--t-h2:clamp(1.6rem,4vw,2.6rem)` / `--t-body:clamp(1rem,1.4vw,1.15rem)` / `--t-hud:.8rem`(고정, tracking .08em).
- 디스플레이 letter-spacing -.02em, HUD/라벨 +.08em 대문자.

## 레이아웃 / 그리드
- 고정 풀블리드 캔버스(z0) + 콘텐츠(z1, max-width 1100px, 좌우 마진 8vw). **포스터-우선 히어로**(문서 아님).
- **비대칭 HUD**(borealis 코너 브래킷 차용): 좌상단 텔레메트리(GEN/FITNESS/CHAMP 수치), 우상단 `◐ SOUND` 토글. 좌하단 스크롤 힌트.
- 콘텐츠 블록은 `.copy`(scrim 플레이트 + backdrop-blur 8px)로 가독 보증.

## 스페이싱
4px 베이스: 4·8·12·16·24·40·64·96.

## 모션 원칙
- 인트로 러시인 1.4s ease-out(2.5s 아님). 상시=느린 패럴럭스 드리프트(전체 회전 금지). 세대 펄스 ~3s 주기 "호흡". 스크롤=강조/도리(dolly) 매핑, 전체 360° 회전 금지.
- **reduced-motion**: 인트로 즉시완료·드리프트0·펄스off·BGM 자동시작 안함. (P5지만 토큰은 지금 정의)
- 이징: `--ease-out:cubic-bezier(.16,1,.3,1)`.

## 사운드
- ambient 우주 드론. 기본 음소거. 우상단 `◐ SOUND` 토글(JetBrains Mono). 첫 토글/제스처로 시작, mute는 localStorage 유지. 입자 펄스↔사운드 동기(옵션).

## 안티-슬롭
보라 그라데이션·3열 아이콘 카드그리드·전체 중앙정렬·장식 블롭 금지. 프로젝트=필드 속 노드(카드그리드 아님). 비대칭 계기판 레이아웃. 포스터, 문서 아님.

# 확정 히어로 방향 (design-shotgun) — **A+C 하이브리드**

베이스 = A "MISSION CONTROL"(계기판/연구실) + 타이포 배치 = C(하단 Space Grotesk 타이틀, HUD와 비경쟁).

## 모션 시스템 ("전부 움직이되 혼잡 금지")
- **원칙:** 느린 ambient 다수 + **동시에 포컬 모션 1개**. 스태거 등장(요소마다 80~150ms 지연). 빠르거나 동시다발 금지(혼잡·멀미 방지). `prefers-reduced-motion` → 전부 정지/즉시완료.
- **텍스트 빌드업:** 헤딩·HUD는 **기계식 등장** — 모노 글자 디코드(랜덤 글자→정착) 또는 타이핑, 픽셀 어셈블. 한 번, 스크롤 진입 시 1회.
- **커서 = 과학 하이라이팅:** 레티클(얇은 십자선)+실시간 좌표 readout(예 `obj[0.62, 0.38]`)+부드러운 스캔 링 펄스가 커서 추종. 입자장에선 borealis식 크레이터 프로브로 연결. 모바일=비활성(터치).
- **GA 데이터-비주얼(연구실):** 중앙 = 계측 차트 — x/y 축 라벨·눈금·옅은 gridline·세대 카운터 틱·fitness 수치 주석이 붙은 2D 산점도 + 빛나는 파레토 호. 세대 진행 시 점들이 새 좌표로 lerp("호흡"), 파레토 호 재계산, 챔피언 타워 갱신. 진짜 진화가 도는 연구 계기.
- **상시:** 입자 미세 드리프트, 파레토 호 맥동, 타워 빔 펄스, HUD 수치 카운트. 전부 저진폭.

## 혼잡 방지 가드(UI/UX)
한 화면에 굵은 모션 1개만 동시. 텍스트 가독 = scrim/backdrop-blur 유지. 모션은 의미 전달용만(장식 과잉 금지). 60fps 예산, 모바일 입자/이펙트 감축.
